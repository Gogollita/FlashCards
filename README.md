# Итоговая аттестационная работа
## Веб-сайт для тренировки лексики в виде игры
Сайт был создан для тренировки лексики в виде игры. 

В части фронтенда использовались основные языки, как HTML, CSS, JavaScript.
В части бэкенда использовался C# и Net. структура. В связи с этим для корректной работы сайта рекомендуется 
запустить систему в папке бэкенда с помощью команды "dotnet run".

Для модели данных используется формат JSON, который регистрирует пользователей и их прогресс.

Сам сайт доступен для устройств от 370px до 1024px (и больше). Все вкладки работают на одной странице. 
Ненужные вкладки скрываются с помощью настройки видимости.

Методология организации CSS - атомарная.
